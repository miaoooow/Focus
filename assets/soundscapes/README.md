# Focus Buddy 精选声音包

本目录只保存由本地原始音频压缩得到的短版 Opus 文件，供桌面版离线播放。
原始文件位于项目根目录下的 `Musics/`，该目录不会提交到 Git 仓库。

当前来源与界面署名：

- The Nature Sounds SocietyJapan — 雨落森林
- The Nature Sounds SocietyJapan — 林间溪流
- Echoes of Nature — 卵石海岸
- The Nature Sounds SocietyJapan — 溪流与鸟鸣

压缩参数：6 分钟、Opus 48kbps、24kHz、单声道。重新分发前请由发布者确认
自己拥有相应音频的使用与再分发权；作者署名不等同于授权证明。
