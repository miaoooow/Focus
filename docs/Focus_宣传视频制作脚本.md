# Focus 宣传视频制作脚本

## 一、成片定位

- 片名：**Focus｜把注意力养成一只猫**
- 受众：18—28 岁学生、初入职场者、自由职业者
- 主版本：60 秒竖屏，1080 × 1920，30 fps
- 衍生版本：15 秒竖屏短版、90 秒横屏演示版
- 核心信息：目标自动解析、网页与软件监督、Luna 成长、声音花园、登录即用免费 AI
- 视觉语气：高级墨绿、奶油白、青柠点缀；安静但不严肃，有一点猫咪式幽默
- 音频建议：前 6 秒使用轻微键盘声与环境底噪；中段进入雨声节奏；完成时加入柔和提示音

## 四、90 秒课程展示版

在 60 秒主片基础上增加以下 30 秒：

1. 10 秒讲解识别原理：扩展读取当前活动域名，EXE 读取前台进程，监测循环不调用大模型。
2. 8 秒讲解安全边界：不截屏、不记录键盘、不读取网页正文。
3. 7 秒讲解降级：云端额度或网络不可用时自动回退本地场景库。
4. 5 秒展示技术结构图：`目标 → 规划 → 白名单 → 监测 → 提醒 → 奖励`。

## 五、必须录制的产品镜头

- GitHub Pages 首页完整打开；
- Focus 扩展“已连接”状态；
- 输入目标并出现本地场景建议；
- 登录后出现 Focus AI 规划；
- 白名单内正常浏览；
- 白名单外持续 8 秒触发提醒；
- 返回任务页后状态恢复；
- 四个声音卡切换；
- 自定义宠物照片的本机模式与云端确认框；
- 完成一轮后的猫币、经验和成长变化；
- Windows 独立窗口与本地软件识别。

录制前把系统缩放设为 100%，浏览器缩放设为 100%，关闭私人通知和无关书签。测试账户、目标文本和宠物照片均使用可公开素材。

## 六、字幕与文案规范

- 每屏不超过两行，每行不超过 14 个汉字；
- 核心字体使用思源黑体或阿里巴巴普惠体；
- 不使用“戒掉”“治愈拖延症”等医疗化或绝对化承诺；
- 不说“完全免费”，改说“提供免费额度”；
- 不说“无需安装”，改说“扩展安装一次，两端共用”；
- 走神提示保持短句，不使用人格攻击。

推荐字幕：

```text
目标自动变成白名单
一个扩展，网页与 EXE 共用
登录即用免费 AI
不截屏，不读取网页正文
把注意力养成一只猫
```

## 七、素材与授权清单

- 小猫图片素材：YouTube 博主 **mocha.**；公开宣传前确认使用与二次分发授权。
- 音频：使用界面中已显示作者的压缩自然声景；确认公开传播许可。
- 宠物上传演示：使用本人拍摄或已取得授权的照片。
- UI 录屏：隐藏用户名、Token、API Key、文件路径和私人窗口标题。

## 八、剪辑与导出清单

- 主色：`#12392C`、`#D7F28F`、`#F1F4E8`
- 字幕安全区：上下各保留画面高度 8%
- 转场：遮罩滑动、卡片推入、轻微景深；避免频繁闪白
- 动效：Luna 以 12—18 帧的小动作穿插，不遮挡核心文字
- 音量：旁白约 -6 dB，背景声约 -22 dB，提示音约 -14 dB
- 导出：H.264，1080 × 1920，12—18 Mbps，AAC 48 kHz
- 封面文字：**把注意力养成一只猫**
- 发布文案：`写下目标，Luna 帮你守住这一轮。Focus 网页与 Windows 版共用一个扩展，登录可使用免费 AI 规划。`
